My (Juang, Yi-Lin's) configurations.

## Dependency
- Bash
- Git
- Tmux
- coreutils: more accurate in command elasped time.
- (Optional) [ESLint](http://eslint.org/): The pluggable linting utility for JavaScript and JSX. Bound with [eslintrc.js](eslintrc.js)
